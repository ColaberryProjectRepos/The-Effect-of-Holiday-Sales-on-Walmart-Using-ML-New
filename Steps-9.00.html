Import CSV file into Power BI Desktop 9
Step 1: Getting the Data set
  1. Download the 'Walmart Sales Data'  file by clicking the red button 
  2. Open Power BI Desktop
  3. Click on Get data button 
  4. Select Text/CSV and click on connect 
  5. Select the 'Walmart' CSV file and click on Open
  6. A preview of the data will be displayed, click on Load
