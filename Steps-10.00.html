Calendar table 10
  1. On Power BI desktop under Modeling click on New table button
  2. Type the Dax formula below in the Dax formula bar, Click on the check mark, and save
Calendar = ADDCOLUMNS(
    CALENDARAUTO(),
    "Year", Year([Date]),
    "Month Number", Month([Date]),
    "Month Name Short", FORMAT([Date], "MMM"),
    "HolidayName",
    SWITCH(
        TRUE(),
        Month([Date]) = 1, "New Years",
        Month([Date]) = 2, "Super Bowl",
        Month([Date]) = 9, "Labor Day",
        Month([Date]) = 11, "Thanksgiving",
        Month([Date]) = 12, "Christmas",
        BLANK()  // Default case if no holiday matches
    )
)


Refresh Date Table
  1. On Power BI Desktop ribbon, Click on Transform data
  2. Click on New Source button, and select Blank query from the list and click on Connect
  3. Under properties in the Name box type in 'Refresh Date'
  4. On the Dax formula bar type ' =Date Time.LocalNow()'
  5. Click on Convert to Table button 
  6. After the refresh table pops up, rename the column by right clicking and choosing rename from the selection
  7. Click  on Close & apply button and save
