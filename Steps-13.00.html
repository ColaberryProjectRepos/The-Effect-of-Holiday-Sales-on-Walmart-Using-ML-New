  1. Create conditional columns for Holiday Order and Unemployment group using steps as shown in the gif and images below


Step: Custom Column Sorting

  1.  Click on the Table tab  on the left of the canvas, and select the 'Walmart' Table
  2. Highlight 'Holiday Flag 2' Column by clicking on the header 
  3. Under Column tools tab, right click on Sort by Column Tab and choose Holiday Order from the drop down option 
  4. Highlight 'Unemployment Rate' column and under sort by column tab, choose Unemployment group
  5. Select Calendar Table, and highlight 'Month Name short' column 
  6. Right Click on Sort by Column Tab and choose Month Number from the drop down option 
  7. Save
