Line and Stacked Column Visuals 17

Steps

  1. Click on an empty space on the canvas, and from the visualizations pane select Line and Stacked Column Chart
  2. Fill up figures provided below for the necessary fields
Build Visual Tab:

  • Shared axis = Year, Month Name Short
  • Column Values = Sales per Month
  • Line Values = Sales YOY, CPI YOY, Holidayinc%
  • Tooltips = Sales YOY,CPI YOY
  • After filling up the above rename fields, as per gif provided below
  • Sort axis by Month Name Short, and ascending order

Format Visual Tab:

               Effects: Background=On, Background color=#E6E6E6, Visual boarder=On, Boarder color-#CCCCCC

Scatter Chart Visual 4(Sales by Store)


Steps
  1. Right click on the sales by Month visual and copy paste (Ctrl+v)
  2. Click on Scatter chart icon from Visuals pane to change the visual in to a scatter chart 
  3. Change figures provided below for the necessary fields
Build Visual Tab:

  • Values = Store
  • X Axis = Weekly Sales
  • Y Axis = Sales YOY
  • Size = Sales per Mth
  • Tooltips = Sales per Mth, Sales YOY, CPI YOY
